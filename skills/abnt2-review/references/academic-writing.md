# Escrita Acadêmica em Português (pt-BR)

## Registro acadêmico

### Impessoalidade

O texto acadêmico deve ser impessoal. Evitar primeira pessoa do singular e plural:

| Errado | Correto |
|---|---|
| `Nós analisamos os dados` | `Os dados foram analisados` |
| `Eu propus um modelo` | `Foi proposto um modelo` / `Este trabalho propõe um modelo` |
| `Nossa pesquisa mostra` | `A pesquisa demonstra` / `Os resultados mostram` |
| `Achamos que` | `Constata-se que` / `Observa-se que` |
| `A gente pode ver` | `Pode-se observar` |

Formas impessoais aceitáveis:
- Voz passiva: `foi desenvolvido`, `foram analisados`
- Pronome `se`: `desenvolveu-se`, `observa-se`, `constatou-se`
- Terceira pessoa com sujeito explícito: `Este trabalho propõe`, `O estudo analisa`

---

### Objetividade e precisão

- **Evitar redundâncias**: `juntamente com`, `totalmente completo`, `absolutamente necessário`
- **Evitar coloquialismos**: `legal`, `bacana`, `tranquilo`, `super`, `mega`
- **Evitar generalizações vagas**: `muitos estudos`, `pesquisadores afirmam` — citar a fonte específica
- **Definir termos técnicos** na primeira ocorrência, especialmente siglas e jargões da área

---

### Coesão e coerência

- **Conectivos de adição**: `além disso`, `ademais`, `outrossim`, `também`
- **Conectivos de causa**: `portanto`, `logo`, `assim`, `desse modo`, `dessa forma`
- **Conectivos de oposição**: `contudo`, `entretanto`, `todavia`, `no entanto`, `porém`
- **Conectivos de conclusão**: `em suma`, `por fim`, `conclui-se que`, `diante do exposto`
- **Evitar**: `então`, `aí`, `mas` para iniciar parágrafos

## Ortografia (Acordo Ortográfico de 2009, em vigor desde 2016)

### Palavras que mudaram com o Acordo

| Antes | Depois |
|---|---|
| `idéia` | `ideia` |
| `heróico` | `heroico` |
| `vôo` | `voo` |
| `zôo` | `zoo` |
| `abóbora` | `abóbora` (sem mudança) |
| `assembléia` | `assembleia` |
| `freqüência` | `frequência` |
| `seqüência` | `sequência` |

### Acento diferencial — removido

- `pôde` (pretérito) vs `pode` (presente): `pôde` **mantém** o acento
- `pôr` (verbo) vs `por` (preposição): `pôr` **mantém** o acento
- Formas como `ele/ela têm` (plural de ter): **mantém** o acento
- `para` (verbo parar), `pelo/pela` (contração): **sem acento**

### Hífen pós-Acordo

| Situação | Regra | Exemplo |
|---|---|---|
| Prefixo + vogal igual | Mantém hífen | `anti-inflamatório`, `micro-ondas` |
| Prefixo + `h` | Mantém hífen | `sub-humano`, `super-homem` |
| Prefixo terminado em vogal + palavra com vogal diferente | Sem hífen | `autoavaliação`, `autoestima` |
| `ex-`, `vice-`, `pré-` | Sempre com hífen | `ex-presidente`, `pré-vestibular` |
| `re-`, `sub-`, `super-` + consoante | Sem hífen | `subproduto`, `reescrever` |

## Gramática frequente em textos acadêmicos

### Crase

**Obrigatória:**
- Antes de substantivo feminino determinado: `à análise`, `à pesquisa`, `às conclusões`
- Antes de locução adverbial feminina: `à medida que`, `à vista`, `à luz de`

**Proibida:**
- Antes de palavra masculina: `ao modelo`, `ao sistema`
- Antes de pronomes: `a ela`, `a você`, `a este`
- Antes de plural indeterminado: `a modelos`, `a pesquisas` (sem artigo "as")
- Antes de verbos: `a partir`, `a analisar`

**Facultativa:**
- Antes de nome de lugar que aceita artigo feminino: `à Teresina` / `a Teresina`

### Concordância verbal com sujeito composto

- Sujeito composto antes do verbo: verbo no plural → `A metodologia e os resultados foram apresentados`
- Sujeito composto após o verbo: verbo concorda com o mais próximo (facultativo) → `Foram analisados os dados e as métricas`

### Regência verbal comum em textos técnicos

| Verbo | Regência correta |
|---|---|
| `visar a` | `O estudo visa **a** melhorar` (não `visa melhorar`) |
| `assistir a` | `assistir **a** uma aula` |
| `obedecer a` | `obedecer **às** normas` |
| `referir-se a` | `referir-se **à** metodologia` |
| `consistir em` | `consiste **em** três etapas` |
| `basear-se em` | `baseia-se **em** dados` |

## Pontuação em contexto acadêmico

### Uso de dois-pontos

Seguido de lista ou explicação:
- `Os resultados indicam: acurácia de 92%, F1-Score de 0,91 e precisão de 90%.`
- Após `:` não se usa maiúscula (exceto nome próprio): `O modelo apresentou: bom desempenho.` ✓

### Ponto final

- Após cada item de lista que forma frase completa
- Após a última palavra-chave do resumo: `algoritmos de aprendizado de máquina.`
- Nas referências: após cada elemento (autor. título. local: editora, ano.)

### Vírgula

- **Obrigatória** antes de orações adverbiais antepostas: `Após o treinamento, os modelos foram avaliados.`
- **Proibida** entre sujeito e verbo: `O modelo **,** apresentou bom desempenho.` ✗
- Antes de "que" relativo com vírgula indica oração explicativa (pode ser removida sem alterar o sentido)

## Estrutura de parágrafos

Um parágrafo acadêmico bem formado tem:

1. **Tópico frasal** (primeira frase): apresenta a ideia principal do parágrafo
2. **Desenvolvimento**: argumentação, dados, citações que sustentam a ideia
3. **Fechamento** (última frase): sintetiza ou conecta ao próximo parágrafo

**Tamanho recomendado:** 3-8 linhas em média. Parágrafos de 1-2 linhas indicam ideia incompleta. Parágrafos de 15+ linhas devem ser divididos.

## Termos e expressões latinas em uso acadêmico

| Expressão | Uso correto |
|---|---|
| `et al.` | Após o primeiro autor quando há 4+; em itálico |
| `apud` | Citação de citação; em itálico |
| `In:` | Separando título de capítulo do livro; maiúsculo, sem itálico |
| `ibid.` / `ibidem` | **Não usar** no sistema ABNTeX2-alf |
| `op. cit.` | **Não usar** no sistema ABNTeX2-alf |
| `sic` | Indica erro no original; entre colchetes: `[sic]` |
| `cf.` | Confrontar/comparar; minúsculo |
